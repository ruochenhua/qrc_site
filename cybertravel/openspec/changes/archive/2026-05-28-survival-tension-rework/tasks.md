# 生存张力重构 — 任务清单

## Phase 1: 数据层重构

- [ ] T01: 定义 `SEASONS` 配置表（四季参数：事件频率、体力倍率、素材加成、食物/住宿倍率）
- [ ] T02: 定义 `WEATHER_TYPES` 配置表（17 种天气效果：体力消耗、素材加成、事件危险度、叙事文本）
- [ ] T03: 定义 `SEASON_WEATHER` 季节专属天气事件池
- [ ] T04: 重构 `ROUTE` 数据结构为 22 节点（短线 8 节点 / 长线 22 节点）
  - [ ] 短线节点配置（成都/邛崃/雅安/天全/二郎山/泸定/折多山/新都桥）
  - [ ] 长线新增节点配置（雅江/卡子拉山/理塘/海子山/巴塘/金沙江/芒康/如美/觉巴山/东达山/左贡/邦达/怒江72拐/八宿/安久拉山/然乌/波密/通麦/鲁朗/色季拉山/林芝/工布江达/墨竹工卡/拉萨）
  - [ ] town 类型节点配置（简化商店+一档住宿）
  - [ ] outpost 类型节点配置（应急商店+无住宿/工棚）
- [ ] T05: 重构 `EVENTS` 事件配置（全部按三分法重写：安全/冒险/黑红）
  - [ ] 8 个大事件重写（爆胎/高原反应/塌方/丢钱包/野狗/赛马节/遇到老手/假剧本邀约）
  - [ ] 10+ 个小事件重写（好天气/下雨/顺风/逆风/日落/平坦/上坡/下坡/饥饿/信号差）
  - [ ] 为所有事件添加 `{weather_text}` 动态文本模板
- [ ] T06: GameState 新增字段
  - [ ] `season`: 当前季节
  - [ ] `weather`: 当前天气
  - [ ] `weatherText`: 当前天气叙事文本
  - [ ] `materialsDay`: 素材采集日期
  - [ ] `lowQualityStreak`: 连续低质量天数
  - [ ] `hypeDecayDays`: 过气 Debuff 剩余天数
  - [ ] `identity`: 角色身份（programmer / laotie）
- [ ] T07: SaveSystem 版本迁移（schemaVersion 3 → 4）
  - [ ] 新增字段默认值填充
  - [ ] 旧存档根据 km 值重新定位 nodeIndex
  - [ ] 旧存档加载提示弹窗（建议新游戏）

## Phase 2: 核心逻辑开发

- [ ] T08: 季节选择系统
  - [ ] 开局流程插入季节选择（路线 → 角色 → 季节）
  - [ ] 季节选择 UI 卡片（四季差异化展示）
- [ ] T09: 每日天气生成系统
  - [ ] `rollDailyWeather(season)` 实现
  - [ ] `getWeatherEffects(weather)` 查询天气效果
  - [ ] 天气倍率与季节倍率乘算叠加
  - [ ] 晨间自动播报（顶部栏 + 日志流）
- [ ] T10: 天气对核心系统的全局影响
  - [ ] 骑行消耗 × 天气体力倍率
  - [ ] 素材获取 × 天气素材加成
  - [ ] 事件危险度加成（影响选项效果或新增危险选项）
- [ ] T11: 事件系统三分法执行层
  - [ ] `EventEngine.executeChoice()` 支持 `type` 字段（safe/adventure/blackred）
  - [ ] 冒险选项收益显著高于安全选项
  - [ ] 黑红选项触发 SinSystem.addSin()
  - [ ] 条件选项（如 `weather_has_rain` 限定）
- [ ] T12: 内容质量评分系统
  - [ ] `calculateContentQuality()` 实现（基础分 + 素材分 + 天气加成 + 季节加成 + 过气 Debuff + 新鲜度加成）
  - [ ] 及格线 60 分判定
  - [ ] <60 分惩罚：全平台掉粉 + SAN -3
  - [ ] 40-59 分低效：粉丝增长 ×0.3
  - [ ] ≥80 分爆款：粉丝增长 ×1.3
  - [ ] 连续 2 天不及格触发"过气" Debuff（3 天质量 ×0.7）
- [ ] T13: 素材保质期系统
  - [ ] 白天事件获得素材时记录 `materialsDay = day`
  - [ ] 夜间结算时检查 `day - materialsDay >= 3`，过期清零
  - [ ] 过期日志提示
- [ ] T14: 程序员专属惩罚落地
  - [ ] 荒野露营时 SAN 额外 -8
  - [ ] 日志文本：【脆皮现充】提示
- [ ] T15: 路线到达逻辑适配
  - [ ] town 类型到达显示简化城市菜单（商店/住宿/离开，无高级选项）
  - [ ] outpost 类型到达显示补给界面（应急商店，工棚住宿可选）
  - [ ] wild_poi 类型到达显示停留拍摄/路过选项
- [ ] T16: outpost 特殊规则
  - [ ] 商店价格系数 ×1.2~×1.5
  - [ ] 工棚住宿恢复系数（energyMult 0.9, conMult 0.8）
