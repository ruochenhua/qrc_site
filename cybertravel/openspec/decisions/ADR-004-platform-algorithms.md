# ADR-004: 三平台差异化算法

## 状态
accepted

## 背景
游戏核心体验是"多平台运营"。不同平台应有差异化算法，迫使玩家做出策略选择，形成博弈深度。

## 决策
设计三个差异化平台，每个平台有独立的算法参数和偏好：

| 平台 | 偏好 | 加成机制 | 惩罚机制 |
|------|------|---------|---------|
| 弹幕网 | 硬核、纯粹 | 高质量内容指数裂变 | 造假被扒皮，精神值雪崩 |
| 短视频网 | 冲突、反转、狠活 | 直播/短视频爆发力极强 | 不整活瞬间取关 |
| 种草网 | 审美至上 | 摄影曝光率高，广告稳定 | 审美不足无流量 |

收益计算公式：
```
fans_gain = base * platform_multiplier * content_quality * (1 + trend_bonus)
cash_gain = fans_gain * platform_cpm * engagement_rate
```

## 后果

### 正面影响
- 丰富的策略选择空间
- 不同流派（视频/摄影/直播）各有优势平台
- 真实反映自媒体平台差异

### 负面影响
- 算法参数需要大量调优
- 平衡性测试工作量大
- 可能过于复杂

### 缓解措施
- MVP 阶段简化算法，使用线性公式
- 通过 playtest 收集数据调参
- 提供算法透明度（玩家可见平台偏好）
