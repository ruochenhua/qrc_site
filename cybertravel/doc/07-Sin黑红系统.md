# CyberTravel · Sin 黑红系统

> 道德抉择机制、弱信号提示、Sin 来源与后果

---

## 一、设计意图

**Sin 系统**是游戏的核心道德张力机制。

在流量至上的网红经济中，"真实记录"往往不如"编造剧本"来得吸引眼球。Sin 系统迫使玩家在以下冲突中做选择：

- **短期利益**：编剧本、消费悲剧、破坏环境 → 快速涨粉、快速赚钱
- **长期代价**：黑红值累积 → 弱信号频发 → 最终身败名裂

Sin 不是简单的"善恶值"，而是**舆论风险值**——反映你的内容在多大程度上建立在虚假和伤害之上。

## 三、弱信号机制

### 概念

当你做了违背良心的事，不会立刻受到惩罚。取而代之的是**弱信号**——一种隐约的不安感，提醒你事情正在失控。

### 触发概率

```
每次增加 Sin 时，40% 概率触发弱信号
```

### 弱信号提示

| Sin 值 | 弱信号示例 |
|--------|-----------|
| 编造剧本 | "💀 评论区出现质疑声..." |
| 虚假宣传 | "💀 粉丝增长突然放缓..." |
| 破坏环境 | "💀 有人私信批评你的行为..." |
| 消费悲剧 | "💀 媒体报道了你的'表演'..." |

**设计意图**：
- 不是直接惩罚，而是制造心理压力
- 玩家可以选择无视（继续 Sin），也可以悬崖勒马
- 弱信号本身不掉属性，但会让玩家感到不安（SAN 隐性压力）

## 四、Sin 状态描述

随着 Sin 累积，游戏会定期提示当前的黑红状态：

| Sin 范围 | 等级 | 提示文案 |
|---------|------|---------|
| 0~9 | - | （无提示） |
| 10~29 | muted | 🌫️ 隐约感觉有些不对劲... |
| 30~49 | info | 💭 有人开始质疑你的内容真实性 |
| 50~79 | warn | ⚡ 争议越来越大，评论区开始失控 |
| 80~100 | danger | ⚠️ 你的黑红指数极高，随时可能被全网封杀 |

**提示频率**：每增加 10 点 Sin 区间，提示一次（通过 flags 去重）。

## 五、Sin 的后果

### 1. 直接结局触发

```
Sin ≥ 80 → 立即触发《黑红之王》结局
```

### 2. 间接影响

| 影响 | 说明 |
|------|------|
| 精神消耗 | 高 Sin 状态下，部分事件 SAN 消耗增加 |
| 粉丝质量下降 | 高 Sin 虽然粉丝数多，但铁粉比例低（预留机制） |
| 好结局阻断 | 抵达拉萨时，Sin ≥ 50 强制触发《黑红之王》而非《抵达圣城》 |

### 3. 角色差异

| 角色 | Sin 特性 |
|------|---------|
| 程序员 | 正常 Sin 增量，容易被舆论击垮 |
| 狠活老铁 | **Sin 减半**，天生适合走黑红路线，可以更大胆地搏流量 |

## 六、Sin 与结局的关联

```
抵达拉萨时：

if 粉丝 > 10万 且 Sin < 20:
    → 《中国骑行第一人》（Good）
elif Sin >= 50:
    → 《黑红之王》（Neutral）
else:
    → 《抵达圣城》（Neutral）
```

| 结局 | Sin 要求 | 粉丝要求 | 类型 |
|------|---------|---------|------|
| 中国骑行第一人 | < 20 | > 10万 | Good |
| 抵达圣城 | < 50 | 任意 | Neutral |
| 黑红之王 | ≥ 50 | 任意 | Neutral |

---

## 七、Sin 系统设计哲学

1. **没有绝对的对错**：Sin 高的玩家可能粉丝更多、赚钱更快
2. **代价延迟**：Sin 的惩罚不是即时的，而是累积的、最终爆发的
3. **角色差异化**：老铁走 Sin 路线有天然优势，程序员走真实路线有优势
4. **弱信号 > 强惩罚**：用心理暗示替代硬惩罚，让玩家自己感到不安
