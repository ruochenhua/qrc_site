# CyberTravel · 结局系统

> 6 种结局的触发条件、优先级、设计意图

---

## 一、结局概览

游戏共有 **6 种结局**，分为 3 类：

| 类型 | 数量 | 结局 |
|------|------|------|
| **Bad（死亡/失败）** | 3 | 横死路边、精神崩溃、破产流浪 |
| **Neutral（普通/争议）** | 2 | 抵达圣城、黑红之王 |
| **Good（完美）** | 1 | 中国骑行第一人 |

---

## 二、Bad 结局（3个）

### 1. 横死路边 💀

| 项目 | 内容 |
|------|------|
| **触发条件** | 体力（CON）≤ 0 |
| **优先级** | 100（最高） |
| **类型** | bad |
| **标题** | 《横死路边》 |
| **描述** | 你的体力在荒野中耗尽，倒在了通往拉萨的路上。风卷着沙尘覆盖了你的自行车。 |
| **典型死因** | 连续拼命赶路、遭遇多次体力惩罚事件、未充分休息 |

**最常见死因**：
- 程序员角色体力初始仅 35，拼命赶路 2-3 次就暴毙
- 高原反应硬撑（体力-15）叠加正常骑行消耗
- 背包超载夜间惩罚累积

---

### 2. 精神崩溃 💀

| 项目 | 内容 |
|------|------|
| **触发条件** | 精神（SAN）≤ 0 |
| **优先级** | 90 |
| **类型** | bad |
| **标题** | 《精神崩溃》 |
| **描述** | 长期的数据焦虑和网暴彻底击垮了你。你删掉了所有视频，消失在网络的另一端。 |
| **典型原因** | 多次遭遇负面事件、Sin 弱信号心理压力、资金紧张 |

---

### 3. 破产流浪 💀

| 项目 | 内容 |
|------|------|
| **触发条件** | 资金（CNY）≤ 0 **且** 总粉丝 < 1000 |
| **优先级** | 80 |
| **类型** | bad |
| **标题** | 《破产流浪》 |
| **描述** | 资金耗尽，粉丝寥寥。你被迫在街头卖艺维生，偶尔有人认出你："这不是那个骑行网红吗？" |
| **典型原因** | 狠活老铁开局资金仅 ¥500，未快速变现涨粉 |

**关键设计**：
- 如果粉丝 ≥ 1000，即使没钱也不会触发此结局（粉丝可以变现养活自己）
- 体现了"流量即生存"的设计主题

---

## 三、Neutral 结局（2个）

### 4. 抵达圣城 ⚖️

| 项目 | 内容 |
|------|------|
| **触发条件** | 抵达最后一个节点（拉萨），且不满足 Good/黑红条件 |
| **优先级** | 50 |
| **类型** | neutral |
| **标题** | 《抵达圣城》 |
| **描述** | 经过漫长的跋涉，你终于看到了布达拉宫的轮廓。没有爆红，但也没有迷失。 |
| **达成难度** | ⭐⭐ |

**条件公式**：
```
if 节点索引 >= 路线长度 - 1
   and 粉丝 <= 10万
   and Sin < 50:
    → 抵达圣城
```

这是**最基础的通关结局**，大多数谨慎玩家会到达这里。

---

### 5. 黑红之王 ⚖️

| 项目 | 内容 |
|------|------|
| **触发条件** | Sin ≥ 80（中途触发）或 抵达时 Sin ≥ 50 |
| **优先级** | 55 |
| **类型** | neutral |
| **标题** | 《黑红之王》 |
| **描述** | 你成了全网最有争议的人。流量爆炸，但每一个评论区都是战场。这真的是你想要的吗？ |
| **达成难度** | ⭐⭐⭐ |

**两种触发路径**：
1. **中途暴雷**：Sin 累积到 80+，直接触发结局（还没走到拉萨就身败名裂）
2. **终点审判**：成功走到拉萨，但因为 Sin 太高，无法获得好结局

**设计意图**：
- 质疑"流量至上"的价值观
- 玩家可能粉丝很多、钱也赚了，但内心空虚

---

## 四、Good 结局（1个）

### 6. 中国骑行第一人 🏆

| 项目 | 内容 |
|------|------|
| **触发条件** | 抵达拉萨 **且** 粉丝 > 10万 **且** Sin < 20 |
| **优先级** | 60 |
| **类型** | good |
| **标题** | 《中国骑行第一人》 |
| **描述** | 硬核的内容、真实的记录让你收获了千万粉丝。你证明了：真实，才是最大的流量密码。 |
| **达成难度** | ⭐⭐⭐⭐⭐ |

**为什么最难达成**：
- 粉丝 > 10万 需要高频优质内容产出
- Sin < 20 意味着必须拒绝所有"走捷径"的诱惑
- 体力/精力/资金三重压力下，还要保持道德纯洁

**策略建议**：
- 程序员角色更适合（高智力 → 视频质量高 → B站涨粉快）
- 白天多拍素材，晚上专注剪视频
- 所有 Sin 事件选"拒绝"
- 保证每天休息恢复，不要拼命赶路

---

## 五、结局优先级总表

当多个结局条件同时满足时，按优先级决定：

| 优先级 | 结局 | 条件 |
|--------|------|------|
| 100 | 横死路边 | 体力 ≤ 0 |
| 90 | 精神崩溃 | 精神 ≤ 0 |
| 80 | 破产流浪 | 资金 ≤ 0 且 粉丝 < 1000 |
| 60 | 中国骑行第一人 | 抵达 + 粉丝 > 10万 + Sin < 20 |
| 55 | 黑红之王 | Sin ≥ 80（中途）或 抵达 + Sin ≥ 50 |
| 50 | 抵达圣城 | 抵达拉萨 |

**检查时机**：
- 每次状态变化后（`checkStatus`）检查死亡/崩溃/破产
- 每天夜间结算后（`checkEnding`）检查终点/好结局

## 六、结局画面

触发结局后，游戏显示：

1. ** Emoji 图标**：Good = 🏆, Bad = 💀, Neutral = ⚖️
2. **标题**：带颜色（Good 绿色、Bad 红色、Neutral 黄色）
3. **描述文案**：情绪化、有画面感的段落
4. **旅程统计**：天数、里程、粉丝、资金、黑红值、终点
5. **结语**：根据结局类型的哲理短句
6. **重新开始按钮**

---

## 七、设计意图

结局系统传达了游戏的核心主题：

> **"流量时代，如何选择成为什么样的人？"**

| 结局 | 主题 |
|------|------|
| Bad（3个） | 身体/心理/经济的崩溃，提醒玩家平衡的重要性 |
| 抵达圣城 | 平凡的完成，大多数人的人生终点 |
| 黑红之王 | 对流量至上主义的讽刺 |
| 中国骑行第一人 | 对"真实即价值"的理想主义肯定 |
