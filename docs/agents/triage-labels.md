# Triage Labels

Canonical label vocabulary for this repo.

| Role | Label | Meaning |
|------|-------|---------|
| Needs evaluation | `needs-triage` | Maintainer needs to evaluate |
| Waiting on reporter | `needs-info` | Waiting for more info from reporter |
| Ready for AFK agent | `ready-for-agent` | Fully specified, an agent can pick it up |
| Ready for human | `ready-for-human` | Needs human implementation |
| Won't fix | `wontfix` | Will not be actioned |

These labels are created on first use if they don't exist.
